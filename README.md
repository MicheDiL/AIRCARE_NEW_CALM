Progetto AIRCARE:
sviluppo del firmware e GUI di controllo del robot CALM ABOVE.
Il folder Qt contiene gli applicativi UI per controllo del sistema e tuning del controllo.
Il folder TI contiene il firmware per il controllo del MCU Hercules RM57.
ATTENZIONE: il driver motore digitale utilizzo in questa versione del FW TI è il drv8231a e non il drv8871.
Il nome del progetto deriva dal fatto che il drv8871 è stato utilizzato per lo sviluppo di un primo prototipo di controllo per un motore DC Maxon.
Il drv8231a è il driver utilizzato per il controllo di attuatori galvanometrici utilizzati in CALM ABOVE.
